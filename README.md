
# 基于rrweb 实现的 网页回放

## 项目结构

- `src/` 源码目录
- `public/` 静态资源目录
- `package.json` 项目配置文件
- `README.md` 项目说明文件
## 项目依赖

- rrweb: 网页录制回放库
- rrweb-player: rrweb 的播放器组件
- rrweb-snapshot: rrweb 的截图库
- rrweb-replay: rrweb 的回放库

### 项目运行

```bash
npm install
npm run dev
```

### 项目

- 采用 monorepo 架构
- 采用 vite 构建
- 采用 vue3 开发
